## To execute

- Build the project:

`$ stack build`

- To execute:

`$ stack exec -- hGeometryCoder <option> <input_file_path> <axis>`

Where:

- option:
  - `-e` : Enconding
  - `-d` : Decoding
  
- input_file_path: input file for the chosen option
- axis: axis throughout the encoding will be made
